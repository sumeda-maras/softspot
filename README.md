# SoftSpot

A dating platform that prioritizes meaningful conversation over appearance-based matching.

## About

SoftSpot enables users to connect through thoughtful prompts and shared values rather than curated photos. The platform uses a gamified progression system where matched users exchange responses to increasingly personal questions, eventually unlocking personalized date suggestions based on shared interests.

## Problem

Dating apps optimize for matches, not conversations. Users struggle to assess compatibility before investing time in conversation, resulting in low first-message rates, high abandonment, and poor retention.

## Solution

- Users answer personality prompts during onboarding
- Matching accounts for relationship intent and prompt responses
- Conversation starts with shared interests instead of generic openers
- Progression system unlocks deeper questions and date recommendations
